# Политика конфиденциальности — Archive5RP Bot

*Последнее обновление: март 2026*

## 1. Общие положения

Настоящая Политика конфиденциальности описывает какие данные собирает Archive5RP Bot, как они используются и хранятся. Используя бота, вы соглашаетесь с данной политикой.

## 2. Какие данные мы собираем

Archive5RP Bot собирает и хранит **минимально необходимый** набор данных:

| Данные | Зачем |
|---|---|
| ID Discord-сервера (`guild_id`) | Для хранения настроек конкретного сервера |
| ID каналов (`channel_id`) | Для отправки уведомлений в нужные каналы |
| ID записей из архива (`cheater_id`) | Для кеша поллинга — чтобы не дублировать уведомления |

**Мы НЕ собираем:**
- Личные данные пользователей (имена, email, пароли)
- Содержимое сообщений
- Историю команд и запросов пользователей
- IP-адреса

## 3. Как используются данные

Собранные данные используются исключительно для:
- Хранения настроек бота на вашем сервере (каналы, параметры мониторинга)
- Предотвращения дублирования уведомлений о новых записях в архиве

## 4. Где хранятся данные

Данные хранятся в базе данных PostgreSQL на серверах **Railway** (railway.app) в регионе Европа. Railway соответствует стандартам GDPR.

## 5. Доступ к данным третьих сторон

Бот взаимодействует с API сайта **archive5rp.ru**. При поиске по нику или Discord ID запрос отправляется на серверы archive5rp.ru. Ознакомьтесь с политикой конфиденциальности archive5rp.ru на их сайте.

## 6. Удаление данных

Если вы хотите удалить все данные вашего сервера:
- Удалите бота с вашего сервера — данные будут удалены по запросу
- Напишите нам напрямую через Discord: **aarchiteector**

## 7. Изменения политики

Мы можем обновлять данную политику. Актуальная версия всегда доступна по этой ссылке. Продолжение использования бота после изменений означает согласие с новой политикой.

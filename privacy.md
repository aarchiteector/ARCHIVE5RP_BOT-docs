# Политика конфиденциальности — Archive5RP Bot

*Последнее обновление: март 2026*

## 1. Общие положения

Настоящая Политика конфиденциальности описывает какие данные собирает Archive5RP Bot, как они используются и хранятся. Используя бота, вы соглашаетесь с данной политикой.

## 2. Какие данные мы собираем

Archive5RP Bot собирает **минимально необходимый** набор данных для работы:

| Данные | Зачем |
|---|---|
| ID Discord-сервера (`guild_id`) | Для хранения настроек конкретного сервера |
| ID канала логов (`mod_channel_id`) | Для отправки уведомлений и отчётов |
| ID канала верификации (`verify_channel_id`) | Для размещения сообщения верификации |
| ID роли верификации (`verified_role_id`) | Для выдачи/снятия роли участникам |
| ID роли для `/scan` (`scan_role_id`) | Для управления доступом к команде сканирования |
| ID записей из архива (`cheater_id`) | Для кеша поллинга — чтобы не дублировать уведомления |
| Включение/выключение функций (`poll_enabled`, `join_check`) | Для хранения настроек мониторинга |

**Мы НЕ собираем:**
- Личные данные пользователей (имена, email, пароли)
- Содержимое сообщений участников
- Историю команд и запросов пользователей
- IP-адреса

## 3. Обработка Discord ID участников

При проверке участников бот использует их **числовой Discord ID** для запроса к API archive5rp.ru. Этот ID:
- Не хранится в нашей базе данных
- Используется только для разового запроса к внешнему API
- Не передаётся третьим лицам кроме archive5rp.ru

## 4. Автоматические действия с данными участников

Бот может выполнять следующие действия на основе ответов API:
- **Кик участников** найденных в базе читеров
- **Выдача роли верификации** прошедшим проверку участникам
- **Снятие роли верификации** при обнаружении в базе

Все эти действия выполняются только на серверах где администратор явно настроил соответствующие параметры через `/setup`.

## 5. Где хранятся данные

Данные хранятся в базе данных PostgreSQL на серверах **Railway** (railway.app) в регионе Европа (europe-west4). Railway соответствует стандартам GDPR.

## 6. Доступ к данным третьих сторон

Бот взаимодействует с API сайта **archive5rp.ru**. При поиске по нику или Discord ID запрос отправляется на серверы archive5rp.ru. Ознакомьтесь с политикой конфиденциальности archive5rp.ru на их сайте.

Мы используем очередь запросов (`apiQueue`) для соблюдения rate-limit API и не передаём никаких дополнительных данных сверх необходимого поискового запроса.

## 7. Удаление данных

Если вы хотите удалить все данные вашего сервера:
- Удалите бота с вашего сервера и напишите нам — данные будут удалены в течение 7 дней
- Напишите напрямую через Discord: **aarchiteector**

## 8. Изменения политики

Мы можем обновлять данную политику. Актуальная версия всегда доступна по этой ссылке. Продолжение использования бота после изменений означает согласие с новой политикой.

# Условия использования — Archive5RP Bot

*Последнее обновление: март 2026*

## 1. Принятие условий

Добавляя Archive5RP Bot на свой Discord-сервер или используя его команды, вы соглашаетесь с настоящими Условиями использования. Если вы не согласны — удалите бота с вашего сервера.

## 2. Описание сервиса

Archive5RP Bot — публичный Discord-бот, интегрированный с базой данных читеров сайта [archive5rp.ru](https://archive5rp.ru). Бот предоставляет:

- Поиск игроков в базе данных по нику или Discord ID
- Автоматические уведомления о новых записях в архиве
- Мониторинг участников сервера при входе по числовому Discord ID
- Просмотр статистики архива

## 3. Правила использования

**Разрешено:**
- Использовать бота для мониторинга и защиты вашего Discord-сервера
- Настраивать бота через команду `/setup`
- Проверять игроков через команды бота

**Запрещено:**
- Использовать бота для преследования, дискриминации или травли пользователей
- Пытаться взломать или нарушить работу бота
- Использовать данные из бота для незаконных целей
- Автоматически массово запрашивать данные через бота

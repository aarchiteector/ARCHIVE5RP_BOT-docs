# Условия использования — Archive5RP Bot

*Последнее обновление: март 2026*

## 1. Принятие условий

Добавляя Archive5RP Bot на свой Discord-сервер или используя его команды, вы соглашаетесь с настоящими Условиями использования. Если вы не согласны — удалите бота с вашего сервера.

## 2. Описание сервиса

Archive5RP Bot — публичный Discord-бот, интегрированный с базой данных читеров сайта [archive5rp.ru](https://archive5rp.ru). Бот предоставляет:

- Поиск игроков по нику или Discord ID
- Автоматическую проверку участников при входе на сервер
- Автоматический кик участников найденных в базе читеров
- Выдачу и снятие ролей верификации
- Полное сканирование всех участников сервера по запросу (`/scan`)
- Фоновый мониторинг: поллинг каждые 5 минут и сканирование раз в 24 часа
- Просмотр статистики архива

## 3. Правила использования

**Разрешено:**
- Использовать бота для мониторинга и защиты вашего Discord-сервера
- Настраивать бота через команду `/setup`
- Использовать команды `/scan`, `/cheater`, `/check_discord`, `/stats`
- Настраивать каналы логов, роли верификации и права доступа к командам

**Запрещено:**
- Использовать бота для преследования, дискриминации или травли пользователей
- Пытаться взломать, нарушить работу или злоупотреблять ботом
- Использовать данные из бота для незаконных целей
- Автоматически массово запрашивать данные через бота
- Использовать функцию автокика без ручной проверки в спорных ситуациях

## 4. Автоматические действия (кик, роли)

Бот может автоматически **кикать участников** и **управлять ролями** на основе данных из базы archive5rp.ru. Администраторы серверов принимают полную ответственность за:

- Включение и настройку функций автокика и верификации

- Ручную проверку спорных случаев

Мы настоятельно рекомендуем проверять уведомления бота перед принятием административных решений.

## 5. Точность данных

Бот отображает данные из базы archive5rp.ru. Проверка осуществляется по числовому Discord ID что обеспечивает высокую точность, однако мы не несём ответственности за точность и актуальность данных самой базы. Все автоматические действия бота основаны исключительно на данных предоставляемых archive5rp.ru.
